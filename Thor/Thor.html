Hi, Welcome to the Thor processor site.

Thor is a work-in-progress.

Thor is a 64 bit cpu with 64 general purpose registers. The Thor guide contains more details on the processor.

 

Features:

- 64 general purpose registers

- 16 code address registers

- 16 predicate registers

- 8 segment registers

- variable length instructions

- predicated execution of all instructions

- load/store architecture , memory mapped I/O

- two addressing modes - register indirect with displacement and indexed

- (2,2) correlating branch prediction

- 2 way fetch, enque, issue, and commit

 

 

Status:

 

- currently the processor is being simulated

- the processor is too large to fit into an inexpensive FPGA

- it's looking like a 32 bit version of the core will just fit in a FPGA

- a GCC backend is in the process of being developed
